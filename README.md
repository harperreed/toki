# Toki

A super simple git-aware CLI todo manager.

## Installation

```bash
go install github.com/harper/toki/cmd/toki@latest
```

## Quick Start

```bash
# Add a todo
toki add "implement feature"

# List todos
toki list

# Mark done
toki done a3f2b9
```

See `docs/plans/2025-11-29-toki-todo-manager-design.md` for full design.
